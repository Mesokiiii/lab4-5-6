# Лабораторные работы №4-6: Анализ трубопроводной сети

WPF приложение для анализа графов трубопроводных сетей с реализацией различных алгоритмов.

## Возможности

### Лабораторная работа №4: Построение графа и алгоритмы обхода
- BFS (обход в ширину)
- DFS (обход в глубину)
- Поиск компонент связности
- **Загрузка пользовательских графов** (TXT, CSV)

### Лабораторная работа №5: Взвешенный граф и алгоритм Дейкстры
- Алгоритм Дейкстры для поиска кратчайших путей
- Расчет расстояний от заданной вершины
- Восстановление маршрута между узлами
- **Загрузка пользовательских взвешенных графов** (TXT, CSV)

### Лабораторная работа №6: Дополнительный анализ
- Поиск точек сочленения (критических узлов)
- Построение минимального остовного дерева (алгоритм Прима)
- Поиск оптимального пути транспортировки

### Эксперимент
- Анализ производительности алгоритмов
- Сравнение времени выполнения на графах разного размера
- **Загрузка больших графов для тестирования**

## Форматы файлов

### Невзвешенный граф (для Lab 4)
```
Узел1 - Узел2
Узел2 - Узел3
Узел3 - Узел4
```

### Взвешенный граф (для Lab 5 и Lab 6)
```
Узел1 - Узел2, 100
Узел2 - Узел3, 85
Узел3 - Узел4, 90
```

Подробнее см. файл `ФОРМАТЫ_ФАЙЛОВ.md`
